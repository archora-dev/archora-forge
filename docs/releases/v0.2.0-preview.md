# Archora Forge v0.2.0-preview Draft

This is a preview/devtool release draft. It is not a production-stable API announcement.

## Highlights

- Runtime auth header presets for bearer and API key flows.
- Remote OpenAPI loading over HTTP(S) with safe header configuration.
- Composition diagnostics and simple safe `allOf` object merge.
- Experimental TanStack Vue Query and Zod generation modes.
- OpenAPI lint and contract-diff CLI commands.
- Experimental plugin hooks for diagnostics and artifact generation.
- Release, docs deploy and beta onboarding assets.

## Known Gaps

- `oneOf`, `anyOf` and discriminator behavior are diagnostic-only.
- Multi-schema generation is not complete.
- Nuxt output conventions are not complete.
- Valibot generation is reserved but not implemented.
- TanStack/Zod modes have isolated generated TypeScript proof, but are still experimental and not complete app integrations.
- Browser-level E2E coverage is still lightweight.
