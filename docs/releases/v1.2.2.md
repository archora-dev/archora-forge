# Archora Forge v1.2.2

Archora Forge v1.2.2 is a patch release for the first-run demo path.

## Fixed

- `archora-forge demo --out <path>` now exits with code `0` after writing the self-contained demo package. The demo can still show a `no-go` decision inside the generated report, but the command itself is a successful walkthrough artifact generator.

## Added

- First-user walkthrough for install, demo and license request.
- Manual license operations notes for trial, pilot, team and organization keys.
- Pilot email kit for the first sales flow.

## Notes

- Use `1.2.2` or newer for public evaluation commands.
- `1.2.1` remains usable, but its demo command can look like a failed command in shell automation when the generated demo decision is `no-go`.
