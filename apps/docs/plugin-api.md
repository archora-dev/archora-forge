# Experimental Plugin API

Forge exposes an internal experimental plugin contract for future adapters and generators. This is not a stable public plugin API.

```ts
import type { ForgePlugin } from '@archora/forge-core'

export const plugin: ForgePlugin = {
  name: 'example-plugin',
  version: '0.1.0',
  diagnostics: (normalized) => [
    {
      severity: 'warning',
      code: 'example-plugin-diagnostic',
      message: `Operations: ${normalized.operations.length}`,
    },
  ],
  generateArtifacts: () => [
    {
      path: 'src/example-plugin.txt',
      content: 'generated by plugin\n',
      kind: 'generated',
      overwrite: true,
    },
  ],
}
```

Hooks currently named in the preview contract:

- `diagnostics`;
- `generateArtifacts`;
- `afterGenerate`.

Diagnostics and generated artifact hooks are wired in this pass. `afterGenerate` runs after files are produced. Schema-normalization hooks are roadmap and are intentionally not part of the beta type contract yet.
