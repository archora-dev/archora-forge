# Roadmap

## Now

- Vue 3.
- TypeScript.
- OpenAPI 3.x local files.
- Schema-derived types.
- Typed client methods.
- Typed query keys and Vue composables.
- Schema-driven forms, tables and pages.
- Routes, permissions and i18n scaffolds.
- Mock fixtures, handlers and scenarios.
- Regeneration-safe generated/custom file policy.
- Local Archora UI fallback adapter.

## Next

- Deeper `oneOf`, `anyOf` and `allOf` modeling.
- Stronger transport layer for auth refresh, retries and typed errors.
- Real external `@archora/ui` package integration once the package contract is available.
- Richer validation metadata.
- More adapters.
- CI mode.
- React target later.

These are roadmap items, not part of the current public-preview scope.
